Hi Andrea,

On your points:
1) Yes — “Begin Game” will send the player directly into the game sequence. And the Denominators link on the main title will navigate to that web page in a separate window so they keep their place in the game.
2) We’re sticking with mouse-only input throughout the game for simplicity and classroom consistency. During the intro animation, clicking will skip it; other inputs are ignored until it finishes and lands on the main title.

Other items you might care about:
- We’ll include a small “Related Projects” area (fractions, decimals, percents). Those links open in a new window on click.
- End-of-game screen will show the player’s exact overall score and a customized message that corresponds to their performance at critical plot points. That screen will offer “Try Again,” “Exit,” and “Main Menu,” all triggered by mouse clicks.
- Scores live only for the current session; when they try again, it resets.

About the content editor/updater tool:
- It will be a simple web form (no installs) for creating and editing questions.
- Forms will use pulldowns and text fields, one page per question, with Next/Prev navigation and a Submit button.
- Submitted data will be validated for completeness (prompt, choices, one correct) and format (fractions in a/b).
- On submit, it will generate a text file on the game server containing the updated question set; the game loads that at runtime.
- You’ll be able to customize per-choice hints/feedback for each question.
- Access to the updater page will be protected with a password so only designated editors can make changes.

We’ll send screenshots once we have the first pass. Feedback welcome.

Thanks,
Priya

Thanks everyone. Quick summary + open items:

- Tech: HTML + linked Flash movies; requires JavaScript and Flash plug-in. No installs on student machines. No direct hardware use.
- Experience: Space Fractions runs standalone in the browser; one interface unifies Intro/Menu/Game/End; external links open in a new window.
- Menu behavior: “Begin Game” goes straight to the game; Denominators link goes to that web page in a new window; Help gives basic instructions.
- Gameplay: mouse-only input. Correct -> show message and advance. Incorrect -> show message and allow retry. Score counts first-try corrects.
- Ending: after the configured number of questions, route to an ending determined by responses at critical questions. End screen shows exact score + customized plot conclusion; options to Try Again / Exit / Main Menu.
- Updater: web form with pulldowns/text fields; page-by-page editing with nav + Submit; validation for completeness/format; generates a text file the game loads at runtime; supports per-choice hints; access protected with a password.
- Availability/consistency: Hosted on S2S website; multiple users acceptable; we’ll test for consistent behavior across supported browsers.
